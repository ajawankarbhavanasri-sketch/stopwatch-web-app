# Elite Stopwatch Pro

A premium Stopwatch Web Application built using HTML, CSS, and JavaScript.

##  Live Demo
 https://endearing-otter-c2aec5.netlify.app/

##  Features
- Millisecond Precision
- Lap Tracking
- Dark Mode
- Responsive Design
- Keyboard Shortcuts
- Export Lap Data
- Modern Glassmorphism UI

##  Technologies Used
- HTML5
- CSS3
- JavaScript
